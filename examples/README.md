# Examples

This folder contains checked-in example council outputs generated by the local CLI.

## Included

- `sample-standard/`: formal alternating rounds plus consensus scaffold
- `sample-live/`: short brainstorm-style exchange scaffold
- `sample-research/`: research-first scaffold with grounded brainstorm and consensus
- `alpha-round-1.md`: tiny manual entry example for `resume`

## What these examples are

These are mostly scaffold outputs, not claims of fully autonomous model execution.

They demonstrate:
- output structure
- JSON state shape
- markdown transcript rendering
- phase naming by mode
- how resume/append fits into the workflow

## What they do not prove

They do not prove:
- automatic OpenClaw session spawning
- autonomous research retrieval
- concurrent agent execution
- parity with Claude Code `--continue`
